# Segundo-Proyecto-Intro-a-progra
Contiene las diferentes versiones y avances realizados para completar el segundo proyecto

Para el correcto funcionamiento del proyecto, se debe crear una carpeta principal, que contenga una carpte llamada "imagenes", en donde se guarden las imagenes utilizadas para el proyecto, esto con el fin de que se carguen correctamente, adempas en la carpeta principal se deben de cargar/ descargar y guardar todos los archivos que esten en el zip, estos estan entrelazados y todos se utilizan para el correcto funcionamiento del programa

Por ultimo se recomienda descargar y utiliar el archivo zip que se subio al respositiorio, este contiene todos los archivos necesarios para el proyecto
